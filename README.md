# belajar-arab
belajar bahasa arab gundul (Baca Gundul)

Prototipe website dan ebook untuk pembelajaran membaca Arab tanpa harakat.

## Menjalankan

Buka `index.html` langsung di browser. Tidak diperlukan instalasi.

Untuk menghasilkan PDF, buka `ebook.html`, klik **Cetak / Simpan PDF**, lalu pilih
printer **Save as PDF** atau **Microsoft Print to PDF**.

## Fitur Unggulan

- **Enam Fondasi Utama** membaca Arab gundul dasar (ditambah Na'at-Man'ut & Idhafah).
- **Materi Interaktif** langsung di website melalui popup materi detail.
- **Kuis Komprehensif** 10 soal interaktif lengkap dengan pembahasan.
- **Progres Belajar** otomatis tersimpan di browser (localStorage).
- **Ebook Siap Cetak** dengan running header & penomoran halaman otomatis.
